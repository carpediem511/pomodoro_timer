## Проект "Таймер 'Pomodoro'"

### ОСНОВНАЯ ИДЕЯ:

Проект представляет собой веб-приложение, реализующее метод Помодоро — эффективный метод управления временем, разработанный для борьбы с прокрастинацией. Цель - создать интерактивный таймер, основанный на методе Помодоро, с функциональностью переключения между  временем работы/отдыха, а также возможностью персонализации интервалов для работы и отдыха.

### ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

1. **Таймер: Отображает обратный отсчет времени для работы/отдыха.

2. **Кнопки "Старт" и "Пауза": Управление запуском и остановкой таймера.

3. **Модальное окно настройки: персонализация временных интервалов для работы и отдыха.


### ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ:

- **Create React App: настройка и сборка проекта.
- **JavaScript/TypeScript: реализация логики таймера и интерфейса.
- **Tailwind CSS: для стилизации интерфейса с применением гибкой системы классов.
- **React-Circular-Progressbar: библиотека для создания динамически изменяемого кругового прогресс-бара вокруг таймера.
- **@mui/material: использование компонента слайдера для настроек в модальном окне.

### ОСОБЕННОСТИ ПРОЕКТА:

-Адаптивный дизайн: разработано с учетом адаптивности под различные устройства.
-Персонализация: пользователь может настроить интервалы времени работы/отдыха.
-Звуковое уведомление: при окончании таймера воспроизводится звуковой сигнал.

### ИНСТРУКЦИИ ПО УСТАНОВКЕ:

1. Склонируйте репозиторий на вашем компьютере.
2. Установите все зависимости, выполнив команду yarn install.
3. Запустите приложение с помощью команды yarn start.
